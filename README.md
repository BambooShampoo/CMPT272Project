# Setup Instructions: 
  
### Initial Setup

Clone Repository -> Switch into CMPT272Project directory

### Install required dependencies
```
npm install
npm install leaflet
npm install react-leaflet
```  

### Start the program
```
npm run dev
```  

### Test program
```
npm run test
```  

### Build application
```
npm run build
```
